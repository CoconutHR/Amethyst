## Install through a package manager
Amethyst is also available in most package repositories

<CodeGroup>

```bash yay
yay -S amethyst-player
```

```bash pacman
pacman -S amethyst-player
```

```powershell winget
winget install -e Geoxor.Amethyst
```

```powershell scoop
scoop install amethyst
```

</CodeGroup>
