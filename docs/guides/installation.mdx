### Installing through a package manager
Amethyst is also available in popular package repositories

<CodeGroup>

```bash yay
yay -S amethyst-player
```

```powershell winget
winget install -e Geoxor.Amethyst
```

```powershell scoop
scoop install amethyst
```

</CodeGroup>

<Note>Installing through a package manager may not always install the latest version</Note>

### Metadata location

Amethyst will create a folder in these locations for storing user-settings, metadata cache and other electron resources

- Windows: `%appdata%/amethyst`
- Linux: `~/.config/amethyst`
- macOS: `~/Library/Application Support/Amethyst`
